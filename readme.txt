== Installation ==


	1. Go to wp-admin -> Plugins > Add New -> and search for 'post to infusionsoft'.
	2. Click to install.
	3. Once installed, activate.
	
OR

Manual Installation:

	1. Upload the entire `post-to-infusionsoft` folder to the `/wp-content/plugins/` directory.
